# my-recipe

### Steps

Новый проект Strapi:
```
strapi new admin
```

Новый проект Vue
```
vue create frontend
```

Axios для работы с API
```
npm install axios
```

Vite для сборки
```
npm install vite
npm install @vitejs/plugin-vue --save-dev
npm run build
npm run dev
```

### Compiles

```
cd admin
npm run develop

cd frontend
npm run dev
```
